An AI-powered Resume & Job Description Analyzer that helps candidates optimize their resumes for Applicant Tracking Systems (ATS). The platform compares resumes with job descriptions using AI and Natural Language Processing (NLP) to provide recruiter-level insights, skill gap analysis, resume improvements, and interview preparation.

---

## ✨ Features

### 📄 Resume Analysis
- Upload Resume (PDF)
- Automatic text extraction
- Resume parsing
- Skills detection
- Experience detection
- Education extraction
- Certifications recognition
- Project extraction

### 💼 Job Description Analysis
- Upload Job Description
- PDF or Text Input
- Extract required skills
- Identify responsibilities
- Detect technical keywords

### 🎯 ATS Resume Matching
- ATS Compatibility Score
- Semantic Similarity Analysis
- Keyword Matching
- Skill Matching
- Resume Ranking
- Recruiter-style evaluation
