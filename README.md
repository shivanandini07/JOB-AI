An AI-powered Resume & Job Description Analyzer that helps candidates optimize their resumes for Applicant Tracking Systems (ATS). The platform compares resumes with job descriptions using AI and Natural Language Processing (NLP) to provide recruiter-level insights, skill gap analysis, resume improvements, and interview preparation.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/jobfit-ai.git
```

Move into the project

```bash
cd jobfit-ai
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file.

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```
